# Task 00 — Python Setup and GitHub Repository

## Goal

The goal of this task is to prepare the development environment and create the main GitHub repository for the training program.

## What You Will Practice

- Installing Python
- Running Python files
- Creating folders
- Using Git
- Creating a GitHub repository
- Writing a basic README file

## Requirements

Create a GitHub repository named:

```text
python-summer-training
```

Inside the repository, create the following folder structure:

```text
python-summer-training/
│
├── task_00_setup/
├── task_01_student_info/
├── task_02_marks_analyzer/
├── task_03_login_system/
├── task_04_guessing_game/
├── task_05_shopping_cart/
├── task_06_contact_book/
├── task_07_calculator_functions/
├── task_08_student_marks_system/
├── task_09_file_handling/
├── task_10_bank_account_oop/
├── task_11_library_oop/
├── task_12_git_teamwork/
├── final_project/
└── README.md
```

## Main README Content

In the main `README.md`, write the following information:

```text
Name:
University ID:
Major:
Why I joined this program:
My goal after finishing this program:
```

## Git Requirements

After creating the folders, push the project to GitHub.

Use at least these commands:

```bash
git init
git add .
git commit -m "Create training repository structure"
git branch -M main
git remote add origin YOUR_REPOSITORY_LINK
git push -u origin main
```

Submit:

- GitHub repository link
- Screenshot of the folder structure
- Screenshot showing that the repository was pushed successfully
